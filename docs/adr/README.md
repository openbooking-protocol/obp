# Architecture Decision Records (ADRs)

This directory contains Architecture Decision Records for the OpenBooking Protocol project.

ADRs document significant technical decisions: what was decided, why, and what alternatives were considered.

## Format

Each ADR is a markdown file named `NNNN-short-title.md`.

## Status values

- **Proposed** — under discussion
- **Accepted** — decision made, being implemented
- **Deprecated** — superseded by a later ADR
- **Superseded by ADR-NNNN** — replaced
